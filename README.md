# TraceAdvisor

The decision layer for serious agronomists. Built for Miraterra.

## Deploy to Vercel (recommended)

1. Upload this folder to a new GitHub repo (drag and drop at https://github.com/new)
2. Go to https://vercel.com → "Add New..." → "Project"
3. Import the GitHub repo
4. Click "Deploy" (no settings to change)
5. You'll get a URL like `trace-advisor.vercel.app`

## Run locally

```
npm install
npm run dev
```

Open http://localhost:5173
